# 🧠 Self-Healing Error Agent

An AI-powered system that detects runtime errors, generates fixes using an agent, validates them with guardrails, and simulates applying patches — with full RCA and timeline visualization.

## 🚀 Features
- Error detection daemon
- Agent-based fix generation
- Guardrails validation
- Patch simulation
- Timeline & RCA dashboard

## 🛠️ Tech Stack
- Backend: Node.js
- Frontend: React
- Hosting: Vercel + Render
- Agent: Simulated / OpenAI (optional)
